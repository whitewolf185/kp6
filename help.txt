Это файл help.txt. В этом файле находится инструкция по эксплуатации БД.
версия 1.2

Зупускается данная программа исключительно на UNIX системах.
Для начала работы нужно запустить консольную программу start.sh с нужными ключами, которые будут приведены ниже.

ОПИСАНИЕ:
Это программа создана для того, чтобы работать с базой данных пассажиров.
У каждого человека есть свои параметры такие как:
    -Фамилия
    -Инициалы
    -Количество багажа (2 символа)
    -Вес всего багажа (3 символа)
    -Место назначения
    -Время вылета (5 символов)
    -Есть ли пересадки у этого пассажира
    -Есть ли у этого пассажира дети
Также программа находит пассажиров, у которых количество сумок больше заданного значения.

СИНТАКСИС:
bash start.sh [КЛЮЧИ] [ПАРАМЕТРЫ] [ПОЛНЫЙ ПУТЬ К ФАЙЛУ]
По умолчанию путь /home/im/CLionProjects/kp6
Также можно сначала создать БД

ключи:
-h --help вызывает данный мануал

-a --add добавляет данные в БД, причем ввод параметров должен быть в виде
[ФАМИЛИЯ] [ИНИЦИАЛЫ] [КОЛ-ВО СУМОК] [ВЕС] [НАЗНАЧЕНИЕ] [ВРЕМЯ] [ПЕРЕСАДКИ(true или false)] [ДЕТИ (true или false)],
причем
    -вес должен вводиться 3-мя цифрами, а кол-во сумок должно вводиться 2-мя цифрами
    -Фамилии с большой буквы
    -Инициалы большими буквами и слитно
    -время в формате 00:00
Пример:
Волков МА 03 042 Марс 04:20 true false ./DB.bin

-q --request на экран выводится первый встретившийся пассажир с заданным параметром,
причем наличие детей и пересадок в данной версии не учитывается

-sa --show_all выводит на экран всю БД

-da --delete_all удаляет всю БД

-d удаляет данные пассажира с соответствующим номером

-с --create создает БД, причем в файл расширения должен быть .bin

-m --make делает соответвущее мне задание, а именно находит пассажиров, у которых количество сумок больше заданного значения.
